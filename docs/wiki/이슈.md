# 이슈

[English](Issues) · **한국어**

이슈는 OrbiTail 에서 일의 단위입니다. 각 이슈는 제목, 상태, 우선순위, 담당자, 날짜, 라벨, 그리고 선택적인 상위 이슈를 가질 수 있습니다. 프로젝트의 이슈는 **테이블**, **보드**, **캘린더**, **타임라인**, **스프린트** 뷰로 볼 수 있으며, 이 페이지에서는 기본 뷰인 테이블을 다룹니다.

## 테이블 뷰

프로젝트를 열고 사이드바에서 **Issues** 를 클릭하면 테이블 뷰에 도착합니다.

![테이블 뷰](https://raw.githubusercontent.com/ruripian/OrbiTail/main/docs/wiki-screenshots/02-table-view.png)

각 행이 이슈이며, ID / 제목 / 상태 / 우선순위 / 담당자 / 시작일 / 마감일 / 라벨이 카드 형태로 표시됩니다. 가능한 조작:

- **컬럼 드래그** — 컬럼 순서 변경.
- **컬럼 리사이즈** — 두 컬럼 헤더 사이의 얇은 구분선을 드래그.
- **컬럼 보이기/숨기기** — 필터 바 우측의 **Columns** 버튼.
- **행을 다른 행 위로 드래그** — 하위 이슈로 중첩. 행 사이로 드래그하면 순서 변경.

## 필터링

테이블 상단의 필터 바에서 상태·우선순위·담당자·라벨 등으로 목록을 좁힐 수 있습니다. 필터 칩을 클릭해 값을 선택하세요.

![필터 드롭다운](https://raw.githubusercontent.com/ruripian/OrbiTail/main/docs/wiki-screenshots/02-table-filters.png)

필터는 AND 로 결합됩니다 — 활성 필터를 모두 만족하는 이슈만 보입니다. 페이지 우상단의 카운터가 실시간으로 매칭되는 이슈 수를 보여줍니다.

## 인라인 편집

테이블의 모든 셀은 곧바로 픽커입니다. 상태·우선순위·담당자·날짜를 직접 클릭하면 그 자리에 드롭다운이 열립니다.

![인라인 상태 픽커](https://raw.githubusercontent.com/ruripian/OrbiTail/main/docs/wiki-screenshots/02-table-inline-edit.png)

값을 고르면 즉시 저장됩니다. 어디서든 **Cmd/Ctrl + Z** 를 누르면 되돌릴 수 있습니다 — undo 스택은 인라인 편집과 벌크 동작의 이전 값을 함께 추적합니다.

## 벌크 액션

각 행 왼쪽의 체크박스로 여러 이슈를 선택할 수 있습니다. 하나라도 선택하면 페이지 하단에 **벌크 툴바** 가 나타납니다.

![벌크 툴바](https://raw.githubusercontent.com/ruripian/OrbiTail/main/docs/wiki-screenshots/02-table-bulk-edit.png)

툴바에서 가능한 액션:

- **Change State** — 선택한 모든 이슈의 상태를 일괄 변경.
- **Change Priority** — 우선순위 일괄 변경.
- **Change Assignee** — 담당자 일괄 재할당.
- **Delete** — 선택한 이슈들을 휴지통으로 이동.
- **Deselect** — 선택 해제.

벌크 변경도 **Cmd/Ctrl + Z** 로 되돌릴 수 있습니다.

## 새 이슈 만들기

테이블 우상단의 **+ Add issue** 버튼, 또는 단축키 **C** 를 누르세요.

![이슈 생성 다이얼로그](https://raw.githubusercontent.com/ruripian/OrbiTail/main/docs/wiki-screenshots/02-issue-create.png)

제목 (필수) 을 입력하고, 선택적으로 우선순위·상태·담당자·날짜·카테고리·스프린트를 설정합니다. 상태는 프로젝트의 기본 상태로 세팅됩니다. **Create issue** 를 눌러 저장하세요.

## 이슈 상세 페이지

테이블에서 이슈 제목을 클릭하면 상세 페이지가 열립니다. 여기서 리치 텍스트 설명·하위 이슈·링크·첨부파일·댓글을 관리합니다.

![이슈 상세](https://raw.githubusercontent.com/ruripian/OrbiTail/main/docs/wiki-screenshots/02-issue-detail.png)

오른쪽 사이드는 모든 편집 필드 (상태, 우선순위, 담당자, 카테고리, 스프린트, 날짜, 라벨, 상위 이슈) 와 주요 액션 버튼을 담습니다:

- **Copy** — 하위 이슈를 포함해 이슈를 복제.
- **Archive** — 보관으로 이동 (되돌릴 수 있음).
- **Delete issue** — 휴지통으로 소프트 삭제. 화면 하단에 **Undo** 버튼이 있는 토스트가 8초간 유지됩니다.

하위 이슈, 링크, 첨부, 댓글, 활동 피드는 상세 페이지 상단의 탭으로 나뉘어 있습니다.
