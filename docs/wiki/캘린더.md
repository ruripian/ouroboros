# 캘린더

[English](Calendar) · **한국어**

캘린더 뷰는 **날짜가 있는 이슈** 와 **독립 이벤트** 를 월 단위 그리드에 함께 보여줍니다. 프로젝트 상단의 탭 바에서 진입할 수 있습니다.

![캘린더 월 뷰](https://raw.githubusercontent.com/ruripian/OrbiTail/main/docs/wiki-screenshots/03-calendar-month.png)

## 이슈 vs 이벤트

- **이슈** 는 시작일부터 마감일까지 이어지는 얇은 바로 표시됩니다. 클릭하면 이슈 상세 페이지로 이동합니다.
- **이벤트** 는 회의·출장·마감·발표·마일스톤처럼 이슈가 아닌 일정을 위한 컬러 블록입니다. 이벤트는 캘린더와 타임라인에만 표시되며 이슈 테이블에는 나오지 않습니다.

날짜 숫자 옆의 작은 점은 해당 날짜에 몇 건이 있는지 알려줍니다.

## 내 일정 필터

캘린더 상단의 **My schedule** 토글은 나와 관련된 항목만 남깁니다.

![내 일정 필터](https://raw.githubusercontent.com/ruripian/OrbiTail/main/docs/wiki-screenshots/03-calendar-my-schedule.png)

**My schedule** 이 켜지면 캘린더는 다음만 표시합니다:

- 내가 담당자인 이슈.
- 내가 참가자인 이벤트.
- **프로젝트 전체 일정** 으로 표시된 이벤트 (모든 멤버가 봐야 하는 일정이므로).

토글을 끄면 프로젝트의 모든 이슈와 이벤트가 다시 보입니다.

## 이벤트 생성과 편집

빈 날짜 칸을 클릭하면 그 날짜의 새 이벤트가 생성되고, 기존 이벤트를 클릭하면 편집창이 열립니다. 둘 다 같은 다이얼로그로 이어집니다.

![이벤트 다이얼로그](https://raw.githubusercontent.com/ruripian/OrbiTail/main/docs/wiki-screenshots/03-event-dialog.png)

설정할 수 있는 항목:

- **Title** — 필수.
- **Start date / End date** — 같은 날이면 단일 일정, 다른 날짜면 기간 일정.
- **Type** — Meeting / Trip / Deadline / Presentation / Milestone / Other. 타입을 바꾸면 기본 색상이 함께 제안됩니다.
- **Color** — 여섯 가지 이벤트 색상 팔레트. 팀 약속에 맞춰 선택.
- **Participants** — 직접 관련된 워크스페이스 멤버.
- **Project-wide event** — 켜 두면 프로젝트의 모든 멤버가 **My schedule** 에서 이 이벤트를 봅니다. 참가자 외에는 상관 없는 회의라면 꺼두세요.
- **Description** — 자유 텍스트.

**Save** 로 기존 이벤트 갱신, **Create** 로 신규 생성, **Delete** 로 삭제할 수 있습니다.
