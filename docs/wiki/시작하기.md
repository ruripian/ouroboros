# 시작하기

[English](Getting-Started) · **한국어**

로그인하면 OrbiTail 이 바로 **워크스페이스 홈** 으로 이동시킵니다. 홈 화면은 이름으로 인사하고, 본인에게 할당된 이슈 수를 보여주며, 워크스페이스 전반의 최근 업데이트를 한눈에 보여줍니다.

![워크스페이스 홈](https://raw.githubusercontent.com/ruripian/OrbiTail/main/docs/wiki-screenshots/01-dashboard-overview.png?v=2)

나머지 기능은 전부 **왼쪽 사이드바** 에서 한 번의 클릭으로 접근할 수 있습니다.

## 사이드바

사이드바는 OrbiTail 의 척추입니다. 위에서 아래 순서로:

![사이드바](https://raw.githubusercontent.com/ruripian/OrbiTail/main/docs/wiki-screenshots/01-sidebar-detail.png?v=2)

- **워크스페이스 전환기** — 상단의 로고와 이름. 클릭하면 소속된 워크스페이스끼리 전환할 수 있습니다.
- **Home** — 나에게 할당된 이슈와 최근 활동이 있는 워크스페이스 홈.
- **Discover** — 워크스페이스 내 공개 프로젝트 중 아직 참가하지 않은 프로젝트 목록.
- **Archived** — 보관된 프로젝트. 참고용으로 남겨두는 곳입니다.
- **Announcements** — 릴리스 노트와 전체 공지. 읽지 않은 항목이 있으면 점(dot) 이 표시됩니다.
- **Projects** — 내 프로젝트. 드래그로 순서 변경, 클릭해 펼치면 프로젝트별 페이지 (Issues, Categories, Trash, Workspace settings) 가 나옵니다.
- **Realtime connected** — 서버와의 WebSocket 연결 상태를 보여주는 실시간 인디케이터. 초록색일 때는 다른 멤버의 변경 사항이 즉시 내 화면에 반영됩니다.

## 상단 바와 사용자 메뉴

모든 페이지의 우측 상단에 있는 아바타를 누르면 개인 메뉴가 열립니다.

![사용자 메뉴](https://raw.githubusercontent.com/ruripian/OrbiTail/main/docs/wiki-screenshots/01-topbar-user-menu.png?v=2)

여기서 **Profile Settings**, **Preferences** (언어, 테마), **Log out** 에 바로 접근할 수 있습니다.
