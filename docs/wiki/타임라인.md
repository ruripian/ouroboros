# 타임라인

[English](Timeline) · **한국어**

타임라인 뷰는 OrbiTail 의 로드맵입니다 — 날짜가 있는 모든 이슈와 이벤트를 연속적인 가로 시간축 위에 플롯하는 Gantt 스타일 차트죠. 프로젝트 내에서 겹침·여유·장기 진행 작업을 한눈에 파악하기 가장 좋은 곳입니다.

![타임라인 개요](https://raw.githubusercontent.com/ruripian/OrbiTail/main/docs/wiki-screenshots/04-timeline-overview.png?v=2)

각 행은 이슈이고, 색상 바가 시작일부터 마감일까지 펼쳐집니다. 한쪽 날짜만 있는 이슈도 단일 날짜 바로 표시되어 빠지지 않습니다. 바에 마우스를 올리면 툴팁으로 제목 전체가 보이고, 클릭하면 이슈 상세 페이지로 이동합니다.

수직 노란 선은 **오늘** 을 표시하며, 스크롤해도 오늘 날짜에 고정됩니다. 각 행 뒤의 은은한 바는 해당 이슈의 스프린트로, 맥락 정보를 한눈에 제공합니다.

## 스케일과 스크롤

그리드 상단에 세 개의 스케일 버튼 — **Day** / **Week** / **Month** — 이 있습니다. 스케일을 바꿔도 현재 보고 있던 시점이 화면 중앙에 유지되어 맥락을 잃지 않습니다. 그리드 안에서 가로로 드래그하면 시간 축을 스크럽할 수 있습니다.

행은 **카테고리** (Design, Engineering, QA…) 단위로 자동 그룹핑되어 관련 있는 작업이 한 덩어리로 모입니다.

## 타임라인 설정

우상단의 **Settings** 버튼을 열면 필터와 표시 옵션이 나옵니다.

![타임라인 설정](https://raw.githubusercontent.com/ruripian/OrbiTail/main/docs/wiki-screenshots/04-timeline-settings.png?v=2)

설정 패널에서 가능한 것:

- **기본 스케일 선택** — Day / Week / Month.
- **행 그룹핑 방식** — Assignee, None, Status, Priority, Category, Sprint.
- **완료된 이슈 표시** — 기본 off. 로드맵이 "아직 진행 중인 일" 에 집중되도록.
- **날짜 없는 이슈 표시** — 일정이 잡히지 않은 작업을 감사할 때.
- **주말 숨기기** — 순수 근무일 기반 계획을 할 때 유용.
- **이벤트 표시** — 캘린더 이벤트를 타임라인 상단에 별도 그룹으로 오버레이.

설정은 사용자별로 저장되어 다음 세션에서도 유지됩니다.
