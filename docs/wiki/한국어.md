# OrbiTail

[English](Home) · **한국어**

OrbiTail 은 Linear 스타일 이슈 트래커, 팀 공유 캘린더, Gantt 형태의 타임라인을 한 곳에 담은 셀프 호스트 프로젝트 관리 도구입니다. Docker Compose 로 내부 인프라에서 바로 구동할 수 있습니다.

![대시보드 개요](https://raw.githubusercontent.com/ruripian/OrbiTail/main/docs/wiki-screenshots/01-dashboard-overview.png)

## 사용자 가이드

- **[시작하기](시작하기)** — 대시보드·사이드바·상단 바 둘러보기
- **[이슈](이슈)** — 테이블 뷰, 벌크 편집, 이슈 상세로 일 관리하기
- **[캘린더](캘린더)** — 팀 일정, 참가자, "내 일정" 필터
- **[타임라인](타임라인)** — 스케일 전환과 설정이 있는 로드맵 뷰
- **[프로젝트](프로젝트)** — 프로젝트 생성, 멤버, 세분화 권한, 보관
- **[공지사항](공지사항)** — 릴리스 노트와 전체 공지
- **[프로필](프로필)** — 개인 설정과 환경 설정

## 스크린샷에 관하여

이 위키의 모든 스크린샷은 **Nimbus Studio** 데모 워크스페이스에서 찍혔습니다. 가상의 모바일 스튜디오로, 세 개의 프로젝트 (Aurora, Meteor, Archive 2025) 와 여섯 명의 멤버가 있습니다. 내 환경에서 동일한 상태를 재현하려면:

```bash
docker compose exec backend python manage.py seed_demo
```
