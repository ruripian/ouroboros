# 프로필

[English](Profile) · **한국어**

개인 설정은 모든 페이지 우상단의 아바타 메뉴에서 접근합니다. **Profile Settings** 를 클릭하면 프로필 편집 화면으로 이동합니다.

![프로필 페이지](https://raw.githubusercontent.com/ruripian/OrbiTail/main/docs/wiki-screenshots/09-profile-page.png)

왼쪽 레일은 설정을 다음 섹션으로 나눕니다:

- **My Account**
  - **Profile** — 이름, 표시 이름, 이메일. 이메일 변경은 새 주소로 전송되는 인증 링크가 필요합니다.
  - **Preferences** — 언어 (영어 또는 한국어), 테마 (System / Light / Dark), 시간대, 주 시작 요일.
  - **Security** — 비밀번호 변경, 활성 세션 보기, 모든 기기에서 로그아웃.
- **Workspace**
  - **Members** — 워크스페이스 전체 멤버 목록 (관리자만).
- **Administration**
  - **User Management** — 신규 가입 승인, staff 권한 부여 (staff 전용).
- **Existing this project**
  - 현재 프로젝트 탈퇴 / 보관 바로가기.

## 언어

OrbiTail 은 영어와 한국어 번역이 함께 제공됩니다. 여기서 고른 언어는 서버에 저장되어 모든 기기에서 동일하게 적용됩니다. 언제든 전환할 수 있으며 페이지 리로드 없이 UI 가 즉시 갱신됩니다.

## 테마

**System** 테마는 OS 의 다크 모드 설정을 따라가며 OS 에서 모드를 전환하면 실시간 반응합니다. **Light** 와 **Dark** 는 OS 설정과 무관하게 고정됩니다. 세 테마 모두 동일한 강조 색상 팔레트를 공유합니다.

## 이름 편집

**First Name** 과 **Last Name** 은 알림 이메일과 활동 피드에 사용됩니다. **Display Name** 은 워크스페이스 안에서 내 아바타 옆에 표시되는 이름입니다. 어느 것을 바꾸든 즉시 반영되며 **Save** 를 누르면 열려 있는 모든 탭에서 아바타 이니셜이 갱신됩니다.
